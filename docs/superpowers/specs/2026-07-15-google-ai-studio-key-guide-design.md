# Google AI Studio API 키 발급 안내 설계

## 목적

간단 설정을 처음 여는 사용자가 Google AI Studio에서 API 키를 발급받아 입력하기까지, 외부 문서를 찾지 않고도 다음 행동을 완료하게 한다.

1. Google 계정으로 Google AI Studio에 로그인한다.
2. 처음 사용하는 경우 API Keys 화면에서 영문 이름의 프로젝트를 만든다.
3. 새 API 키를 만들고 복사한다.
4. 생성한 프로젝트의 결제 등급이 `무료 등급(Free)`인지 확인한다.
5. 생성된 API 키를 간단 설정의 입력란에 붙여 넣고 `API 키 확인` 버튼으로 연결 확인 결과를 확인한다.

## 범위

- 간단 설정의 API 키 입력과 기존 Google AI 안내 문구 아래에만 `Google AI Studio에서 API 키 받기` 단계 안내를 둔다. 고급 설정에는 이 안내를 렌더링하지 않는다.
- 단계 안내는 실제 Google AI Studio 흐름에 맞춰 로그인, 처음 사용하는 경우의 영문 프로젝트 생성, 키 생성·복사, 무료 등급 확인, 키 복사·입력의 다섯 단계로 구성한다.
- 기존 `Get API Key` 링크와 더미 YouTube 설정 가이드 링크는 유지한다.
- `Get API Key`를 포함한 간단 설정의 라벨은 현재 UI 언어에 맞게 표시한다. 한국어를 원본으로 영어·일본어 메시지를 같은 키 구조로 동기화한다.
- API 키 입력은 한 개만 유지한다. 입력 시 `gemini-3.1-flash-lite` 설정은 자동 저장하고, 연결 검증은 `API 키 확인` 버튼을 눌렀을 때만 실행한다.

## 화면과 문구

- 탭 라벨은 한국어에서 `간단 설정`, `고급 설정`을 사용한다.
- 간단 설정 제목은 `Google Gemini로 무료 자막 번역 시작하기`, 입력 라벨은 `Google AI API 키`로 표기한다.
- 가이드 제목은 `Google AI Studio에서 API 키 받기`로 한다.
- 처음 사용하는 경우의 프로젝트 생성 단계에는 이름을 `Gemini Project`처럼 영어로 입력하도록 안내한다.
- 키를 만든 다음에는 생성한 프로젝트의 결제 등급이 `무료 등급(Free)`인지 확인하도록 안내한다.
- 무료 등급 안내에는 사용 한도가 있으며, 일일 요청 한도는 미국 태평양 시간 자정에 초기화된다는 점을 명시한다.
- 가이드의 마지막 단계는 키를 붙여 넣으면 `Gemini 3.1 Flash Lite`가 자동 설정되고, `API 키 확인` 버튼으로 연결을 확인한다는 결과를 명시한다.
- API 키 입력 아래에 `API 키 확인` 버튼을 두고, 고급 설정의 연결 테스트처럼 성공·실패 결과를 같은 상태 메시지 영역에 표시한다.
- 키는 비밀번호처럼 취급해야 하므로, 다른 사람과 공유하거나 공개 문서에 넣지 말라는 짧은 주의 문구를 포함한다.

## 오류 처리와 보안

- API 키를 만들 권한이 없으면 사용 중인 Google Cloud 프로젝트의 관리자에게 권한을 요청하거나 새 개인 프로젝트를 사용하도록 안내한다.
- 연결 검증에 실패하면 기존 실패 메시지와 현재 번역 Provider 유지 동작을 그대로 사용한다.
- 저장 뒤 API 키는 기존 마스킹 규칙에 따라 표시하고, 단계 안내에는 키의 전체 값을 다시 보여주지 않는다.

## 검증

- 옵션 레이아웃 테스트로 간단 설정 가이드의 제목·다섯 단계·무료 등급 일일 한도 안내·`API 키 확인` 버튼·주의 문구·두 링크를 확인한다.
- 한국어·영어·일본어 메시지 파일이 동일한 키를 가지며, 링크 라벨까지 현지화되는지 확인한다.
- 기존 간단 설정 동작 테스트와 전체 `npm test`, `npm run check`를 실행한다.
