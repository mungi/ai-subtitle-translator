# AST Hover-Only Submenu 설계

## 목적

AST 메뉴의 `원본 자막`과 `번역 스타일` 서브메뉴를 마우스 hover에서만 열리게 하고, 상위 항목 클릭으로 열린 상태를 고정하거나 해제하는 기능을 제거한다.

## 범위

- 상위 항목 클릭으로 수정하던 두 확장 상태와 `.open` 클래스를 제거한다.
- CSS는 `:hover`에서만 서브메뉴 목록을 표시한다. 클릭으로 남는 포커스의 `:focus-within` 표시도 제거한다.
- 하위 원본 자막과 번역 스타일 항목의 선택 클릭은 그대로 유지한다.

## 검증

- 상위 항목을 클릭해도 메뉴를 다시 렌더링하거나 열린 클래스를 추가하지 않는다.
- CSS에 hover 표시 규칙은 남고 `:focus-within`과 `.open` 표시 규칙은 없다.
- 기존 자막·스타일 선택 및 전체 content-script 테스트를 실행한다.
