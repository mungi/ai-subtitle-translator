# 컴팩트 번역 스타일 이름 표시 설계

## 목적

설정 화면에서는 번역 스타일의 전체 설명을 유지하고, 공간이 제한된 확장 아이콘 팝업과 AST 메뉴에서는 `Natural`, `Lecture`, `Technical`, `Custom 1`, `Custom 2`처럼 설명 앞의 이름만 표시한다.

## 범위

- `extension/popup/popup.js`의 팝업 요약 표시
- `extension/content/content-script.js`의 AST 번역 스타일 메뉴 토글과 선택 항목
- 설정 화면, 저장되는 `translationStyle` 값, 시스템 프롬프트에는 변경을 가하지 않는다.

## 구현과 검증

각 표시 경로에서 로컬라이즈된 스타일 문구를 `공백-하이픈-공백` 기준으로 한 번 나누고 앞부분만 사용한다. 문구에 구분자가 없으면 전체 문구를 유지한다. 팝업과 AST 메뉴가 컴팩트 표기를 사용하고 옵션 화면이 기존 전체 문구를 계속 사용하는지 테스트한다.
