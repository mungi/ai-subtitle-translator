AI Subtitle Translator

AI provider를 선택하면 LLM이 영상 전체의 흐름을 반영해 자막을 번역합니다. 사용자 프롬프트로 원하는 말투와 번역 스타일도 직접 지정하세요.

WHAT IT DOES

자막을 더 편하게 읽는 방법

01

영상 위에서 바로

Udemy와 YouTube 플레이어 위에 번역 자막을 표시합니다. 컨트롤 바의 아이콘으로 언제든 켜고 끌 수 있습니다.

02

영상 전체를 이해하는 번역

AI provider를 선택하면 LLM이 앞뒤 자막과 영상의 흐름을 함께 고려해, 끊기지 않고 자연스러운 번역을 만듭니다.

03

원하는 말투로

사용자 프롬프트로 강의체, 친근한 말투, 기술 문서 스타일처럼 원하는 번역 톤과 표현 방식을 직접 설정할 수 있습니다.

GET STARTED

네 단계로 시작하세요

  1. 1확장 프로그램을 설치합니다.
  2. 2Options에서 목표 언어와 번역 provider를 설정합니다.
  3. 3Udemy 강의 또는 YouTube 영상을 엽니다.
  4. 4영상 컨트롤 바의 자막 번역 아이콘을 누릅니다.

MODEL SELECTION

작고 빠른 모델부터

자막 번역은 명확한 입출력 작업이므로 가장 큰 프론티어 모델이 필요하지 않은 경우가 많습니다. Gemini 3.1 Flash-Lite, GPT-5.6 Luna, Claude Haiku 4.5 같은 작고 빠른 모델부터 사용하세요.

PAID RECOMMENDATION

OpenRouter + DeepSeek V4 Flash

유료 번역을 위한 가성비 시작점입니다. Options에서 OpenRouter provider를 선택하고 `deepseek/deepseek-v4-flash` 모델을 입력하세요.

PRIVACY & SECURITY

번역에 필요한 데이터만 사용합니다

광고·추적·분석을 위한 개발자 서버를 운영하지 않습니다.

자막 텍스트와 번역 설정은 사용자가 선택한 번역 provider로 전송될 수 있습니다. API key와 설정, 번역 캐시는 사용자 브라우저에 저장되며 개발자 서버로 전송되지 않습니다.

API key는 암호화해 저장하고 content script에서 직접 읽을 수 없게 제한합니다. 다만 자동 복호화에 필요한 정보도 같은 브라우저 프로필에 있으므로 OS 보안 저장소와 같은 수준의 vault는 아닙니다.
