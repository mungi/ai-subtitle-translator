AST - AI Subtitle Translator

AI provider를 선택하면 LLM이 영상 전체의 흐름을 반영해 자막을 번역합니다. 사용자 프롬프트로 원하는 말투와 번역 스타일도 직접 지정하세요.

WHAT IT DOES

자막을 더 편하게 읽는 방법

01

영상 위에서 바로

Udemy, YouTube, TED, NVIDIA Academy, Vimeo 플레이어 위에 자막을 표시합니다. AST 메뉴에서 자막을 켜고 끄고, 원본 자막 언어와 번역 스타일을 바로 선택할 수 있습니다.

02

영상 전체를 이해하는 번역

AI provider를 선택하면 LLM이 앞뒤 자막과 영상의 흐름을 함께 고려해, 끊기지 않고 자연스러운 번역을 만듭니다.

03

원하는 말투로

사용자 프롬프트로 강의체, 친근한 말투, 기술 문서 스타일처럼 원하는 번역 톤과 일타 강사 스타일 처럼 표현 방식에 대한 프롬프트를 직접 설정할 수 있습니다.

아이콘 색상으로 현재 번역 상태를 바로 확인할 수 있습니다. 자막 또는 번역 요청을 준비하는 동안에는 현재 색상으로 미세하게 깜빡입니다.

  • 흰색 · 기본AST가 꺼져 있거나 원문 자막 상태
  • 파란색 · 활성번역 자막을 준비하거나 표시하는 기본 활성 상태
  • 노란색 · 임시 번역LLM 최종 번역을 기다리는 동안 임시 번역 표시
  • 초록색 · 현재 cue 완료현재 재생 cue의 최종 번역 준비 완료
  • 보라색 · 전체 완료전체 자막의 최종 번역 완료
  • 분홍색 · 대체 번역쿼터 초과 등으로 Google Translate 대체 번역 사용 중

  1. 1확장 프로그램을 설치합니다.
  2. 2Options에서 목표 언어와 번역 provider를 설정합니다.

PRIVACY & SECURITY

번역에 필요한 데이터만 사용합니다

광고·추적·분석을 위한 개발자 서버를 운영하지 않습니다.

자막 텍스트와 번역 설정은 사용자가 선택한 번역 provider로 전송될 수 있습니다. API key와 설정, 번역 캐시는 사용자 브라우저에 저장되며 개발자 서버로 전송되지 않습니다.

API key는 암호화해 저장하고 content script에서 직접 읽을 수 없게 제한합니다. 다만 자동 복호화에 필요한 정보도 같은 브라우저 프로필에 있으므로 OS 보안 저장소와 같은 수준의 vault는 아닙니다.
